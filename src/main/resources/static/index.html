Secure Asset Access API

A Spring Boot demonstration service for managing technical assets, role-based permissions, and immutable-style audit records. Built as portfolio evidence for Java backend and application-security roles.

Checking service health...

Role-based API access

HTTP Basic authentication with separate ADMIN, OPERATOR, and VIEWER permissions.

Asset grant registry

Admins record VIEW or OPERATE grants for named principals. This demonstrator stores grants for audit review; runtime grant enforcement is intentionally out of scope.

Audit trail

Asset creation and access grants are persisted as dated audit events, readable by administrators only.

Endpoint contract

MethodEndpointRequired rolePurpose
POST/api/assetsADMINCreate a classified technical asset
GET/api/assetsVIEWER / OPERATOR / ADMINList assets
POST/api/assets/{id}/access-grantsADMINRecord a named grant and audit event
